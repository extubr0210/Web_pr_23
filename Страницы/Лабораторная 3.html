Пункт 1. css правила @media. Запись "@media(){}" по сути можно представить себе как "if(){}" Содержимое будет выполняться когда условие истино. Содержимым могут быть как несколько свойств внутри селектора так и несколько селекторов.
После слова @media можно указать тип устройства (all,print,screen,speech). Обычно используется @media screen.
Также в условии можно указать требуемые характеристики.

Например, цвет текста этого предложения будет меняться в зависимости от ширины экрана.
Также можно скрывать элементы, например когда ожидается что они не поместятся в клиентскую область.(Это предложение исчезнет когда текст сверху станет красным)

Пункт 2. Позиционирование. Для использования позиционирования мы используем свойство position. Виды позиционирования:

  • static - статическое. По умолчанию.
  • relative - относительное. С фиксированным смещением относительно статического.
  • absolute - абсолютное. В фиксированном месте ближайшего позиционированного предка.
  • fixed - фиксированное. В фиксированном месте относительно области просмотра.
  • sticky - липкое. Как относительное пока страница не прокручена до определеннного порога, затем как абсолютное.

Пункт 3. Таблица (с ,например, стилями для четных и нечетных строк).

Существует группа псевдоклассов (псевдоклассы используются внутри css селекторов) "child" позволяющая выбирать элементы по порядковому номеру внутри элемента.

  • first-child - первый дочерний элемент
  • last-child - последний дочерний элемент
  • nth-child() - каждый элемент номер которого удовлетворяет выражению (вида a*n+b) в скобках.
  • nth-last-child() - каждый элемент номер которого удовлетворяет выражению, но номера считаются с конца.

Чтобы получить четные ,нечетные (и много какие ещё) элементы мы используем nth-child().
Чтобы получить таблицу с разностилизованными четными нечетными строками, нам стоило бы использовать тег "table", и написать что то вроде "tr:nth-child(even)" для четных строк, и "tr:nth-child(odd)" для нечетных. Но мы же не ищем лёгких путей. Но ничего не мешает нам сделать это и для других элементов со свойством display:grig .
