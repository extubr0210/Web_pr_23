Пункт 2 Добавление JS в html.
Тег "script" отвечает за скрипты. Скрипт выполняется немедленно в том месте где он находится в разметке. Поэтому его положение важно. В моменте выполнения скрипт не увидит любой разметки находящейся ниже него. Поэтому принято подключать js скрипты в самом конце кода, перед закрывающим тегом body.
Атрибут src подволяет подключать js файл в место где находится тег script.

Пункт 3.1 Вывод информации.
Вывод в консоль - console.log()
Вывод с назойливым сообщением - window.alert()
Вывод через перезапись куска разметки - .innerHTML

Пункт 3.2 (немедленно вызываемая функция).
В JS есть функции. Но в JS функция может быть значением переменной, и передаваться как аргумент.
После слова "function" язык ожидает объявление функции. Поэтому для IIFE нам нужно шаманить с синтаксисом чтобы "function" рассматривалась языком в качестве выражения. Например добавив перед "function" унарный оператор, void, или обернуть выражение скобками.
Пример iife:
(function(){
alert("some text");
})();

Пункт 3.3 события
Когда события происходят, скрипты могут активироваться. Назначить обработчик события можно:
Через атрибут элемента (например onclick)
Через свойство DOM объекта (например onclick)
Подписавшись на событие с помощью addEventListener.
