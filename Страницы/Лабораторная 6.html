Объект js.
Вспомним что содержимым переменной в js могут быть данные_простых_типов, функции , объекты.
Объект js - это ("массив","карта") несколько пар "ключ"-"значение" . Значениями также могут быть данные_простых_типов, функции, объекты, массивы.

JSON.
Это существует такое расширение файла "*.json" . JSON и объект_js можно преобразовать друг в друга.
JSON - . Содержимым могут быть данные_простых_типов, объекты, массивы. (Но не функции (методы) )

API (application programming interfaces). Расшифровка абривиатуры является описанием достаточной точности. Обычно для доступа к функционалу api создаётся новый экземпляр соответствующего объекта, затем могут быть использованы методы этого объекта.

Promise. Это такой объект. Создаётся с помощью " * = new Promise(f);" В качестве аргумента промису передаётся функция которую мы "обещаем" выполнить. Эта функция, в свою очередь, обязана принимать в качестве своих 2х аргументов 2 встроенные в js функции "resolve" и "reject".

Then. Метод объекта promise. Имеет вид: "*.then(f1,f2);". В качестве аргументов принимает 2 функции, одна из них выполнится, какая именно- зависит от того успешно ли завершился промис из которого then был вызван. (Функции должны принимать единственный аргумент: "result" или "error" соответственно.)

Fetch. Функция. Возвращает promise объект. Синтаксис: "*= fetch(url,[options]);". По умолчанию результатом успешного выполнения такого промиса будет файл найденный по указанному адресу.
