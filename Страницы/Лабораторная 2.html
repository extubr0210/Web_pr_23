Страница с лабораторной 2

Обратно на стартовую страницу


Пункт 1. Подключили файл стилей "Лабораторная 2 стили.css" в дополнение к "стили общие.css".

Пункт 2.1 Границы элемента задаются с помощью свойства border. Можно задать толщину границы, её стиль и цвет.
Пункт 2.2 Отступы внутренние задаются с помощью padding. Отступы внешние задаются с помощью margin. Можно задать отступы для каждой из 4 сторон несколькими способами (для_всех_сторон),(вверх_низ,право_лево),(вверх,право_лево,низ),(верх, право, низ, лево)
Пункт 2.3 Шрифты задаются свойством font. Подробнее в пункте 6.

Пункт 3.1 Типы селекторов.

  • Селекторы тегов. (h2)
  • Селекторы классов. (.something)
  • Селекторы идентификаторов. (#something)
  • Селекторы атрибутов. (a[some_atribut])На наличие атрибута. Или (a[some_atribute="value"]) на значение атрибута.
  • Селекторы псевдокласов. (a:hover) Для стилизации определенного состояния элемента.
  • Селекторы псевдоэлементов. (p::first-line) Для стилизации части элемента.
  • Сложные селекторы(Прочие селекторы).

Сложные селекторы (могут включать)

  • Группировка. (селектор_1 , селектор2) ИЛИ
  • Комбинирование. (селектор_1селектор_2) И
  • Вложенность. (селектор_1 селектор2) селектор_2 "которые находятся в" селектор_1
  • Дочернесть. (селектор_1 > селектор_2) селектор_2 "которые являются непосредственными потомками" селектор_1
  • Следующего элемента. (селектор_1 + селектор_2) селектор_2 "которые расположенны сразу после" селектор_1
  • Последующего элемента. (селектор_1 ~ селектор_2) селектор_2 "которые расположены после" селектор_1

Распространённые методологии селекторов

БЭМ: Мы используем селекторы классов.
SMACSS: Правило называть css правила на основе их классификации на 5 категорий (base,layout,module,state,theme)
ECSS: Мы используем селекторы идентификаторов.

Пункт 4. Картинки добавляются при помощи тега img. Его атрибуты: src (ссылка на файл картинки);alt (Описание содержимого картинки в текстовом виде)

Пункт 5.1 Цвет заливки фона задаётся с помощью свойства background-color.

Пункт 5.2 Тени задаются с помощью свойства box-shadow.

Пункт 6

Пункт 6.1 Размер шрифта задаётся свойством font-size.
Пункт 6.2 Высота строки задаётся свойством line-height.
Пункт 6.3 Семейство используемых шрифтов задаётся свойством font-familly.
Пункт 6.4 Насыщенность шрифта задаётся свойством font-weight.
Пункт 6.5.1 Выравнивание текста задаётся свойством text-align.
Пункт 6.5.2 Вертикальное выравнивание задаётся свойством vertical-align.

Пункт 6.6.1 Цвет заливки фона смотри пункт 5.1

Пункт 6.6.2 Цвет текста задаётся свойством color.
Пункт 6.7 Отступы в тексте задаются свойством text-indent.

Пункт 6.8 Пробелы в тексте обрабатываются при помощи свойства white-space.

Пункт 6.9 Преформатированный текст задаётся свойством pre.

Пункт 6.10 Подчеркивание, зачеркивание и другие. Определяются высшими силами. Определяются свойством text-decoration.
Пункт 6.11 Регистр символов задаётся свойством text-transform.
